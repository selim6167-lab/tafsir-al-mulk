# Comparative Tafsir of Surah Al-Mulk (67)

A verse-by-verse comparative analysis of Surah Al-Mulk drawn from 8 classical and modern tafsir sources.

## Sources

| Tafsir | Author | Approach |
|--------|--------|----------|
| تفسير ابن كثير | Ibn Kathir (d. 774 AH) | Hadith-based, narration-heavy |
| تفسير القرطبي | al-Qurtubi (d. 671 AH) | Fiqh rulings + linguistic analysis |
| الكشاف | al-Zamakhshari (d. 538 AH) | Deep balagha (rhetoric) + i'rab |
| تفسير البغوي | al-Baghawi (d. 516 AH) | Concise athar-based |
| تفسير الماتريدي | al-Maturidi (d. 333 AH) | Hanafi aqidah perspective |
| التحرير والتنوير | Ibn Ashur (d. 1393 AH) | Maqasid + modern linguistic analysis |
| التفسير المنير | Wahbah al-Zuhayli (d. 1436 AH) | Structured modern: vocab, grammar, rhetoric, rulings |
| صفوة التفاسير | al-Sabuni (d. 1442 AH) | Accessible modern summary |

## Structure

Each ayah follows this format:

- **Core Meaning**: Synthesized paragraph drawing the best from all 8 sources
- **Linguistic Analysis**: Key vocabulary, grammatical points, rhetorical devices
- **Points of Divergence**: Where scholars disagree, with named positions
- **Unique Insights**: Notable points from individual mufassirun

## Source Data

Raw tafsir texts were extracted from the Shamela Library (المكتبة الشاملة) database containing 7,872 books.
